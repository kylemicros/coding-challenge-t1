# User Management Service

Take-Home Assignment Challenge

## Introduction

This is the initial attempt in creating a **User Management Microservice** for A SaaS platform. It showcases a simple approach by following a feature-based architecture, where all related files are grouped in one place, improving the developer experience (DX) and future maintenance and scalability.

## Tech Stacks

The following are the technologies used in this project:

- NestJS
- TypeORM
- MySQL 8
- Redis
- Docker
- Docker Compose
